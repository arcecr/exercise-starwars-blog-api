# StarWars API with Flask

> For 4Geeks Academy

## Installation (automatic if you are using Gitpod)

```bash
$ pipenv install
$ mysql -u root -e "CREATE DATABASE example"
$ pipenv run init
$ pipenv run migrate
$ pipenv run upgrade
```
